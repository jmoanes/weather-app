Weather App
===========

A simple weather application that allows users to search for cities and view weather information.

Features:
---------
- Search for cities and view current weather
- View a 5-day weather forecast
- Responsive design for desktop and mobile
- Add and remove favorite cities

How to Run Locally:
-------------------
1. Install dependencies:
   npm install
2. Start the development server:
   npm start
3. Open your browser and go to:
   http://localhost:1234

Project Structure:
------------------
- src/index.html         # Main HTML file
- src/js/                # JavaScript source files
- src/styles/            # CSS stylesheets
- assets/                # Images and assets

Deployment:
-----------
You can deploy this app to GitHub Pages or any static hosting service. For GitHub Pages:
1. Push your code to a GitHub repository
2. Enable GitHub Pages in the repository settings
3. Set the source branch and folder (usually 'main' and '/root' or '/docs')
